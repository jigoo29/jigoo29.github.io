Repository addon

Instructions for adding this repo:

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://jigoo29.github.io/repo/ (Give it the name "jp repo").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "JP repo", and install repository.JPB-1.04.zip
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "JP Repo"
  • Go into the "add-ons" section in the repo, and you'll find JP repo

Enjoy!
